# nvm

https://github.com/nvm-sh/nvm

```shell
nvm install node
```

Always default to the latest available node version on a shell

```shell
nvm alias default node
```

```shell
nvm alias default v9.3.0
```

Use the latest available 8.0.x release

```shell
nvm use 8.0
```
